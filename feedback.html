Suggest a tool to assess

Know a tool we should look at?

If you come across an AI tool that might be useful for teaching, assessment, or research at WMS, flag it here. The bar is low — a name and a line on what it does is enough. We review every suggestion. Assessed tools are added to the external tools page with a full structured assessment. If a tool is assessed and added to the hub, the person who flagged it will be credited.

A useful suggestion includes: tool name and link, what it does (one or two sentences), which part of the WMS workflow it might fit (marking, feedback, research, teaching, student support), and optionally your initial impression if you've tried it.
